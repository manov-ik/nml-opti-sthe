### Final Year Project
## NUMERICAL AND MACHINE LEARNING APPROACH TO OPTIMIZE SHELL AND TUBE HEAT EXCHANGER IN CHEMICAL INDUSTRIES

This project focuses on optimizing the performance of a shell and tube heat exchanger (STHE) using a hybrid approach that combines Computational Fluid Dynamics (CFD) and Machine Learning (ML). Initially, a detailed 3D CAD model of the heat exchanger is developed and analyzed using CFD simulations to study fluid flow behavior, temperature distribution, and pressure drop under various operating conditions.

The simulation data generated from multiple parametric studies is used to build a dataset linking design parameters with performance outputs such as heat transfer rate and pressure loss. A Machine Learning model, specifically a Random Forest Regressor, is trained on this dataset to predict thermal performance accurately without the need for repeated simulations.

The integrated CFD - ML framework enables faster evaluation of design configurations and supports optimization by identifying parameter combinations that maximize heat transfer efficiency while minimizing pressure drop. This approach significantly reduces computational cost and provides a scalable solution for heat exchanger design in chemical industries.
